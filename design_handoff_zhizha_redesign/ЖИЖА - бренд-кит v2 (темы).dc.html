Один бренд, четыре характера

ЖИЖА осталась той же — округлой, тёплой, чуть дерзкой. Поменяли только цвет: вместо одной сладкой палитры теперь система из четырёх тем. Жмякай свотчи вверху — весь документ переоденется на лету. То же самое увидит пользователь в меню тем.

Вольт — по умолчанию Карамель — наследие Магма Хром

Меняет тема

  • Фирменный градиент и акцентные цвета
  • Тон тёмной базы и поверхностей
  • Цвет текста и свечения (glow)
  • Цвет текста поверх акцента (тёмный/белый)

Не трогает никогда

  • Шрифты — Comfortaa + Nunito + JetBrains Mono
  • Tone of voice — на «ты», глагол вперёд, коротко
  • Радиусы, отступы, тени, пружинистое движение
  • Структуру и поведение компонентов

Правило простое: тема — это только цветовой слой. Если правка трогает форму, шрифт или текст — это не тема, это изменение всей системы.

01 · Темы

Четыре настроения

Каждая тема — своя сигнатура из 3 цветов на тёмной базе. Карточки показывают палитру независимо от активной темы.

Вольт

дефолт

Кибер-электрик. Дерзко, технологично, заметно.

Карамель

наследие

Та самая первая. Мягко, сладко, по-домашнему.

Магма

Тёплый огонь. Жарко, энергично, но не приторно.

Хром

Холодный графит и один электрик-синий. Чётко, строго.

04 · Форма — общая для всех тем

Округлость и пружина

Никаких острых углов: всё интерактивное — радиус ≥ 12px. Чем крупнее элемент, тем больше скругление.

Тени и свечение

07 · Приложение

Код тем — готов к вставке

Общие токены (форма, движение) в :root, цветовой слой — в блоках [data-theme]. По умолчанию активна Вольт.

/* ОБЩЕЕ для всех тем — форма, движение, шкала */
:root{
  --r-xs:8px; --r-sm:12px; --r-md:16px; --r-lg:22px; --r-xl:28px; --r-pill:999px;
  --shadow-panel:0 20px 60px rgba(0,0,0,.55);
  --shadow-card:0 8px 24px rgba(0,0,0,.35);
  --ease-goo:cubic-bezier(.34,1.56,.64,1);
  --ease-soft:cubic-bezier(.4,0,.2,1);
  --dur-1:120ms; --dur-2:200ms; --dur-3:320ms; --dur-4:600ms;
}

// применение и сохранение темы
function setTheme(name){
  document.documentElement.dataset.theme = name;   // 'volt' | 'karamel' | 'magma' | 'chrome'
  localStorage.setItem('zhz-theme', name);
}
// на старте: сохранённая тема или Вольт по умолчанию
setTheme(localStorage.getItem('zhz-theme') || 'volt');
