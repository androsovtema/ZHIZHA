# ЖИЖА — бренд-кит и дизайн-система

> Версия 1.0. Это источник правды по визуальному языку ЖИЖА: голос, шрифты, цвета,
> отступы, компоненты, шаблоны и правила. Меняешь оформление продукта — сверяйся отсюда,
> а не из головы.

ЖИЖА — это не «генератор фонов на WebGL». ЖИЖА — это **живая жижа**: тёплая, текучая,
цветная, чуть дерзкая. Старое оформление (системный шрифт, сине-фиолетовый SaaS-градиент,
строгий стекломорфизм) выглядело как любой дашборд. Новый язык должен с первого взгляда
говорить: «это мягкое, живое и быстрое».

Три принципа, из которых растёт всё остальное:

1. **Холст — герой, интерфейс — тень.** Фон, который делает пользователь, всегда ярче UI.
   Панель — тёмное матовое стекло, она не спорит с картинкой.
2. **Мягко и округло.** Никаких острых углов. Скругления щедрые, движения пружинят,
   шрифт с круглыми терминалами. Всё как капля жижи.
3. **Быстро и по-дружески.** Creator пришёл за результатом за 20 секунд. Текст короткий,
   на «ты», технарь спрятан в «Профи».

## 1. Tone of voice — голос бренда

**Архетип:** Шут + Творец. Друг-дизайнер, который за минуту накидывает тебе огонь-фон
и подмигивает.

**Характер:** живой · щедрый · быстрый · чуть дерзкий · никогда не душный.

### Правила речи

- **На «ты».** «Выбери», «покрути», «скачай» — не «выберите».
- **Глагол вперёд.** Интерфейсные подписи начинаются с действия.
- **Коротко.** Кнопка — 1–2 слова. Подсказка — одна строка.
- **Без технаря на первом экране.** «GLSL», «октавы fbm», «domain warp», «curl-flow»
  живут в блоке «Профи». На свету — человеческие слова: «движение», «искажение», «зерно».
- **Лёгкий юмор, ноль кринжа.** Можно подмигнуть, нельзя натужно шутить.
- **Эмодзи — почти нет.** Максимум один и по делу. ЖИЖА выражается цветом и формой, не смайлами.

### Камертон

| Ситуация | ✅ Голос ЖИЖА | ❌ Не ЖИЖА |
|---|---|---|
| Заголовок | Живые фоны за 20 секунд | Процедурный генератор фонов на WebGL |
| Кнопка экспорта | Скачать видео | Инициировать экспорт |
| Пустое состояние | Тут пока пусто. Тыкни в шаблон 👇 | Нет выбранных пресетов |
| Подсказка | Помедленнее — фон под текст должен дышать, а не мельтешить | Рекомендуемое значение параметра speed: 0.2–0.3 |
| Ошибка записи | Видео не записалось. Chrome или Edge справятся лучше | RecorderError: codec unsupported |
| Успех | Готово. Лови файл | Операция успешно завершена |

### Словарь (rename для UI)

| Технично | По-человечески |
|---|---|
| Field type / pattern | Стиль |
| Surface / halftone / ASCII | Фактура |
| Domain warp | Искажение |
| Grain | Зерно |
| Speed | Скорость / Движение |
| Tileable | Бесшовный |
| Render-on-demand | (не показываем) |

## 2. Шрифты

Два округлых шрифта в паре + моноширинный для кода. Оба основных — с полной кириллицей,
бесплатные (Google Fonts).

### Акцентный / дисплейный — **Comfortaa**

Геометрический гротеск с **круглыми терминалами** — буквально «жижа в типографике».
Логотип ЖИЖА, крупные заголовки, ярлыки шаблонов, числа-герои.

- Начертания: 500 / 700.
- Только крупно (≥ 18px). Мелкий текст Comfortaa теряет читаемость — для него Nunito.
- Трекинг для логотипа: +0.5px. Для заголовков: 0.

```
@import url('https://fonts.googleapis.com/css2?family=Comfortaa:wght@500;700&display=swap');
```

### Интерфейсный / основной — **Nunito**

Гуманистический гротеск с мягко скруглёнными формами. Вся панель: подписи, кнопки,
значения, подсказки, бодикопи. Отлично читается на 10–14px.

- Начертания: 400 / 600 / 700.
- Базовый размер UI: 13px. Микроподписи: 11px (не мельче).

```
@import url('https://fonts.googleapis.com/css2?family=Nunito:wght@400;600;700&display=swap');
```

### Моноширинный — **JetBrains Mono**

Только для технического: embed-код, URL сцены (`#s=…`), поле импорта.

```
font-family: 'JetBrains Mono', ui-monospace, Menlo, monospace;
```

### Типографическая шкала

| Токен | Размер / интерлиньяж | Вес | Шрифт | Где |
|---|---|---|---|---|
| display | 40 / 1.05 | 700 | Comfortaa | Лендинг-герой, big hook |
| h1 | 22 / 1.15 | 700 | Comfortaa | Логотип в панели, экраны |
| h2 | 16 / 1.2 | 700 | Comfortaa | Названия шаблонов |
| label | 11 / 1.2 | 700 | Nunito | Заголовки секций (UPPERCASE, +1.2px трекинг) |
| body | 13 / 1.5 | 400 | Nunito | Основной текст панели |
| value | 13 / 1.2 | 700 | Nunito | Значения слайдеров (tabular-nums) |
| caption| 11 / 1.4 | 400 | Nunito | Подсказки, мелочь |
| code | 12 / 1.4 | 400 | JetBrains Mono | Технические поля |

## 3. Цвет

Тёмная база (чтобы фон-холст сиял), а вместо корпоративного сине-фиолетового —
фирменный **многоцветный жижа-градиент**: розовый → фиолетовый → мята. Это сигнатура.

### Чернила / тёмная база

| Токен | HEX | Применение |
|---|---|---|
| `--ink-900` | `#0A0912` | Фон приложения (тёмный с лёгким фиолетовым уходом) |
| `--ink-800` | `#12101D` | Заполнение карточек/секций |
| `--ink-700` | `#1A1726` | Ховер-поверхности |
| `--panel` | `rgba(22,18,38,.55)` | Матовое стекло панели (+ `backdrop-filter: blur(22px)`) |
| `--panel-border` | `rgba(255,255,255,.10)` | Границы |
| `--panel-hi` | `rgba(255,255,255,.06)` | Внутренние подсветки/секции |

### Текст

| Токен | HEX | Применение |
|---|---|---|
| `--text` | `#F4F1FF` | Основной (тёплый белый, лёгкий фиолет) |
| `--text-dim` | `#A39FB8` | Вторичный |
| `--text-mute` | `#6E6A86` | Подсказки, плейсхолдеры |

### Фирменный градиент «Жижа»

Главный цветовой жест. Кнопки-действия, активные состояния, заполнение слайдеров,
логотип, тоглы.

```css
/* основной — 3 стопа, для крупных плоскостей и логотипа */
--zhizha-glow: linear-gradient(135deg, #FF5FA8 0%, #8A5CFF 55%, #2FE6C8 100%);
/* компактный — 2 стопа, для мелких кнопок (контрастнее) */
--zhizha-grad: linear-gradient(135deg, #8A5CFF 0%, #FF5FA8 100%);
```

Акцентные solid-цвета (из тех же трёх точек):

| Токен | HEX | Роль |
|---|---|---|
| `--accent` | `#8A5CFF` | Фиолетовый — основной акцент, фокус-кольца |
| `--accent-pink` | `#FF5FA8` | Розовый — co-акцент |
| `--accent-mint` | `#2FE6C8` | Мята — co-акцент / success |

### Семантика

| Токен | HEX | Роль |
|---|---|---|
| `--success` | `#2FE6C8` | Готово, успех |
| `--warning` | `#FFB13D` | Внимание |
| `--danger` | `#FF5C6E` | Ошибка, удаление |
| `--info` | `#8A5CFF` | Инфо |

### Правила цвета

- **Один градиент в кадре** среди UI-элементов рядом. Два жижа-градиента впритык дерутся.
- **Градиент — только для действия и активности.** Не красить им контейнеры и фоны секций.
- **Текст по градиенту** — белый `#FFFFFF` при достаточном контрасте; иначе тёмные чернила
  `#0A0912`. Контраст текста на градиенте/жиже держим ≥ 4.5:1, при нужде добавляем скрим.
- Палитры самого холста (Сияние, Закат, Океан…) — это **контент**, а не UI-токены. Не тянуть
  их в интерфейс.

## 4. Отступы, скругления, тени, движение

### Spacing — шкала 4px

`2 · 4 · 8 · 12 · 16 · 20 · 24 · 32 · 40 · 48 · 64`

- Внутренние отступы контролов: 8–12.
- Паддинг карточек/секций: 12–16.
- Паддинг панели: 16.
- Промежутки между секциями: 12.
- Воздух вокруг блоков-героев: 24–32.

### Радиусы (щедрые, «жижа»)

| Токен | px | Где |
|---|---|---|
| `--r-xs` | 8 | Свотчи, чипсы-теги |
| `--r-sm` | 12 | Инпуты, мелкие кнопки |
| `--r-md` | 16 | Карточки шаблонов, секции |
| `--r-lg` | 22 | Панель, крупные карточки, модалки |
| `--r-xl` | 28 | Hero-блоки |
| `--r-pill`| 999 | Чипсы-категории, тоглы, primary-кнопки |

Правило: чем крупнее элемент, тем больше радиус. Минимум для всего интерактивного — 12px.
Никаких прямых углов на кнопках и карточках.

### Тени и свечение

```css
--shadow-panel: 0 20px 60px rgba(0,0,0,.55);          /* парящая панель */
--shadow-card:  0 8px 24px rgba(0,0,0,.35);            /* карточки */
--glow-accent:  0 8px 28px rgba(138,92,255,.35);       /* фиолетовое свечение под action */
--glow-mint:    0 8px 28px rgba(47,230,200,.30);       /* для success */
```

Свечение (`glow`) — фирменный приём: action-элементы будто светятся изнутри, как жижа.
Применять умеренно: primary-кнопка, активная карточка.

### Движение — текучее и пружинистое

```css
--ease-goo:  cubic-bezier(.34, 1.56, .64, 1);  /* лёгкий овершут — пружинит как капля */
--ease-soft: cubic-bezier(.4, 0, .2, 1);       /* плавно, без отскока */
--dur-1: 120ms;  /* микро: ховеры */
--dur-2: 200ms;  /* стандарт: появления контролов */
--dur-3: 320ms;  /* карточки, модалки */
--dur-4: 600ms;  /* крупные переходы, морфинг */
```

- Ховеры и нажатия — `--ease-goo`, чтобы «отскакивало».
- Нажатие кнопки: `transform: scale(.97)`.
- Появление карточек/панелей — `--ease-soft`.

## 5. Компоненты — короткая спека

Все компоненты наследуют токены выше. Состояния: default / hover / active / disabled / focus.

### Кнопка — primary (action)

- Фон `--zhizha-grad`, текст `#fff` 700, радиус `--r-pill`, паддинг 10×18.
- Hover: `--glow-accent` + `translateY(-1px)`. Active: `scale(.97)`.
- Disabled: убрать градиент → `--ink-700`, текст `--text-mute`.
- Фокус: кольцо `0 0 0 3px rgba(138,92,255,.4)`.

### Кнопка — ghost / secondary

- Фон `rgba(255,255,255,.03)`, граница `--panel-border`, текст `--text`, радиус `--r-sm`.
- Hover: фон `rgba(255,255,255,.08)`.

### Чипс-категория (pill)

- Радиус `--r-pill`, паддинг 7×13, текст 12px `--text-dim`.
- Активный: фон `--zhizha-grad`, текст `#fff`.

### Карточка шаблона

- Радиус `--r-md`, граница `--panel-border`, превью-тумба сверху (живой градиент шаблона).
- Hover: `translateY(-2px)` + `--shadow-card`.
- Активная: граница `--accent`, внутреннее кольцо `inset 0 0 0 1px var(--accent)` + лёгкий glow.
- Мета: название — Comfortaa 700 13px; муд — Nunito 400 11px `--text-dim`.

### Слайдер

- Трек: `rgba(255,255,255,.10)`, заполнение: `--zhizha-grad`.
- Бегунок: круг 16px, фон `--zhizha-grad`, тонкая белая обводка, `--glow-accent` на hover.

### Тогл

- Off: `rgba(255,255,255,.12)`. On: `--zhizha-grad`. Бегунок белый, переход `--ease-goo`.

### Лого в панели

- Иконка-капля (`assets/icon-180.png`) 34px, радиус `--r-sm`, мягкая тень `--glow-accent`.
- Рядом «ЖИЖА» — Comfortaa 700, трекинг +0.5px.

## 6. Шаблоны внутри ЖИЖА

Шаблон ЖИЖА — это **сохранённая сцена** (тип поля + фактура + палитра + движение + формат),
поданная под понятную задачу creator'а. Чтобы 16+ шаблонов смотрелись и разнообразно, и как
одна семья, держим их в **мудах** и общих диапазонах параметров.

### Анатомия шаблона

```js
{
  id:    'soft-gradient',          // kebab-case, англ., техническое
  name:  'Мягкий градиент',        // 1–2 слова, русское, образное
  mood:  'спокойно · мягко',       // ровно 2 тега через « · »
  family:'calm',                   // муд-семья (см. ниже)
  ratio: '9:16',                   // дефолтный формат под задачу
  v:     [speed, zoom, warp, grain, fieldType, palette, surface, pixel, halftone, tiles, ...]
}
```

### Муд-семьи

Каждая семья = задача creator'а + диапазон параметров. Новый шаблон обязан попадать в семью.

| Семья | Задача | Скорость | Искажение (warp) | Зерно | Фактура | Палитры | Примеры |
|---|---|---|---|---|---|---|---|
| **calm** — спокойные/под текст | Фон под текст, цитаты, говорящая голова | 0.15–0.30 | 1.5–3.0 | низкое | мягкая/нет | пастель: Сумерки, Мята, Ирис | Бархат, Страты, Послесвет, Мягкий градиент |
| **energy** — клиповые | Реелс, тикток, хайп-вставки | 0.6–0.95 | 5.0–8.0 | средн. | острая/нет | яркие: Закат, Уголь, Сияние | Плазма, Магма, Неоновый импульс, Кислотный клип, Рябь |
| **deep** — атмосферные | Глубина, обложки, интро | 0.35–0.5 | 6.0–8.0 | низкое | мягкая | Океан, Ирис, Сияние | Бореалис, Туманность, Прилив |
| **graphic** — структурные | Графичный бренд-фон, паттерн | 0.3–0.45 | 2.0–5.0 | низкое | гекс/ASCII/линии/полутон | контраст: Уголь, Хром, Океан | Плетение, Контур, Решётка, Оникс, Гидрополе |
| **ornament** — симметрия | Медитативное, мандалы, луп | 0.3–0.4 | 4.0–6.0 | низкое | мягкая | Ирис, Сияние, Мята | Мандала, Подсолнух, Калейдо |

### Правила именования шаблонов

- **Одно образное русское слово** — явление, материал, природа: *Бархат, Магма, Прилив, Страты*.
- **Без английского и без технаря** в `name`. «Curl-flow» → нет. «Течение» → да.
- Имя передаёт **ощущение/фактуру**, а не функцию. Не «Быстрый фон» — а «Кислотный клип».
- `mood` — ровно два тега-прилагательных/наречия через « · »: `быстро · ярко`, `тихо · читаемо`.
- `id` — латиница kebab-case, можно техническое.

### Чек-лист создания нового шаблона

1. **Семья.** Определи муд-семью. Параметры — внутри её диапазонов (таблица выше). Иначе семья «расползётся».
2. **Палитра из набора.** Бери из палитр семьи. Кастомный градиент — только если он реально новый муд.
3. **Формат под задачу.** `calm`/`energy` → 9:16, `graphic`/`deep` → 16:9, `ornament` → 1:1.
4. **Фактура к месту.** Гекс/ASCII/полутон только в `graphic`. В `calm` фактура убивает читаемость текста — не ставить.
5. **Зерно умеренно.** > среднего только в `energy`.
6. **Тест на читаемость.** Если шаблон под текст (`calm`) — положи мысленно строку Comfortaa 700: видно?
7. **Имя по правилам** (см. выше) + два муд-тега.
8. **Превью-тумба** отражает реальную палитру шаблона.
9. **Проверь соседей.** В сетке рядом не должно быть двух почти одинаковых — нужен видимый контраст муда.

### Разнообразие, но семья

Чтобы сетка выглядела богато и собранно одновременно:

- В каждой видимой группе — **минимум 3 разных семьи**.
- Не больше **2 шаблонов на одну палитру** подряд.
- Чередуй «спокойное ↔ яркое» — глаз не должен залипать на однородном.
- Связку держат **общие токены**: одинаковые радиусы тумб, типографика меты, ховеры. Разнятся
  только сами картинки.

## 7. Примеры постов (как creator применяет фон)

ЖИЖА отдаёт фон — поверх него creator кладёт текст/объекты. Эти раскладки показывают, как
бренд-кит работает «в бою», и задают правила оверлея.

### Общие правила оверлея текста по жиже

- **Скрим под текст.** Поверх живого фона — затемнение 0–45% (линейный градиент снизу или
  радиальный под блоком), иначе текст тонет. Особенно для семьи `energy`.
- **Хук — Comfortaa 700**, крупно. Подзаголовок/тело — Nunito 600/400.
- **Текст под фон → бери семью `calm` или заморозь** (Статика) и убери искажение. Не клади
  текст на самую «кипящую» зону.
- **Контраст ≥ 4.5:1.** Светлый фон → тёмные чернила `#0A0912`, тёмный → `#FFFFFF`.

### Safe-zones по форматам

| Формат | Площадка | Держать текст | Избегать |
|---|---|---|---|
| 9:16 | TikTok / Reels / Stories | центральная треть | верх ~130px (аватар), низ ~240px (подпись, UI) |
| 1:1 | пост-карусель, подкаст | центр, поля 80px | самые края |
| 16:9 | YouTube-обложка | левая ⅔ под текст | правый-низ (таймкод) |

### Раскладки-рецепты

**A. Цитата в Stories (9:16, семья `calm`)**
Фон: Бархат / Сумерки, Статика. Скрим 25% снизу. По центру — цитата Comfortaa 700 ~34px
белым, ниже автор Nunito 600 14px `--accent-mint`. Воздух 64px по краям.

**B. Обложка Reels (9:16, семья `energy`)**
Фон: Кислотный клип / Закат, движение. Радиальный скрим под хуком. Хук Comfortaa 700 ~40px
в 2–3 строки, ключевое слово — заливка `--zhizha-grad` по тексту. Низ свободен под подпись TikTok.

**C. Превью YouTube (16:9, семья `deep`)**
Фон: Бореалис / Океан. Текст слева Comfortaa 700, плашка-пилюля `--zhizha-grad` с цифрой/темой.
Правый низ пустой.

**D. Обложка подкаста (1:1, семья `ornament`)**
Фон: Мандала / Ирис, Статика. По центру — название Comfortaa 700, обводка-пилюля. Симметрия
работает на «иконичность».

**E. Графичный бренд-фон под оверлеи (16:9, семья `graphic`)**
Фон: Гидрополе / Хром, полутон. Спокойный графичный паттерн под логотип/UI-наложения,
не перетягивает внимание.

## 8. Правила использования бренд-кита

### Логотип

- **Чистое поле** вокруг лого = высота буквы «Ж» со всех сторон.
- **Минимальный размер** иконки-капли — 24px; слово «ЖИЖА» — не мельче 14px (Comfortaa 700).
- Слово ЖИЖА — **только Comfortaa 700**, трекинг +0.5px. Не заменять другим шрифтом, не курсивить, не растягивать.
- Иконка может светиться (`--glow-accent`), но **не перекрашивать** саму каплю произвольными цветами.

### Цвет

- Жижа-градиент — **для действия и активности**, не для фонов/контейнеров.
- **Не больше одного** жижа-градиента среди близко стоящих UI-элементов.
- Не смешивать два разных градиента в одном элементе.
- Палитры холста (Океан, Закат…) — контент, **не использовать как UI-цвета**.
- Тёмная база обязательна для приложения: фон-холст должен быть ярче интерфейса.

### Типографика

- Comfortaa — **только крупно** (≥18px) и для акцентов/заголовков/лого. Не пускать в мелкий UI.
- Весь мелкий текст и тело — **Nunito**. Моно — только для кода/URL.
- Не мельче 11px. UPPERCASE — только для label-заголовков секций (+трекинг).
- Не более двух гарнитур в кадре (+ моно для кода).

### Форма и движение

- Никаких острых углов: всё интерактивное — радиус ≥ 12px.
- Ховеры пружинят (`--ease-goo`), действия слегка «вдавливаются» (`scale(.97)`).
- Glow — умеренно и только под action/активное.

### ❌ Анти-паттерны

| Не делай | Почему |
|---|---|
| Системный шрифт в заголовках | Теряется «жижа», получается дашборд |
| Сине-фиолетовый старый акцент `#6ea8ff/#b07bff` | Заменён фирменным жижа-градиентом |
| Острые углы, радиус < 12 на кнопках | Противоречит мягкости бренда |
| Технарь на первом экране («GLSL», «fbm») | Пугает creator'а — прячь в «Профи» |
| Два градиента впритык / градиент на фоне секции | Визуальный шум, акцент размывается |
| Текст по «кипящему» фону без скрима | Нечитаемо |
| Палитра холста как цвет кнопки | Путаница контента и UI |
